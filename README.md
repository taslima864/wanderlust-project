# wanderlust-project
Wanderlust is a full-stack travel accommodation platform inspired by Airbnb. Users can explore destinations, view property listings, add new places, write reviews, and manage listings with authentication and authorization.

This project demonstrates real-world full-stack development, REST APIs, database design, authentication, and responsive UI.
